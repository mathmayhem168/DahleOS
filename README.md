Welcome to DahleOS! It's a fun little operating system I've been working on, and it uses QEMU. 
Dahle is basically my own language with tons of grammar rules. It's not actually going
to be official, though, like my OS. 


---


## What is DahleOS?
DahleOS is a small operating system that I run on MacOS. It's nothing official though.

## Further Description about DahleOS
DahleOS uses:
| Name | Purpose |
|---|---|
| QEMU | Used for emulating the computer |
| Assembly files | Assembling the core kernel |
| C files | Very low-level files that control the applications |
| Compilation tools | Helps compatibility with all operating systems and helps compile assembly |
| Makefile | Simplifies the stuff you need to type in to run the operating system |
| .bin files | Binary |
