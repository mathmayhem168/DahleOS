# CONTRIBUTING.md of DahleOS

## Introduction

Hello, and welcome to DahleOS, my little operating system!
Glad you're here. Here are some tips before you dive in.

---

## Editing

### Important Tips

1. **Don't touch the assembly.** It's fragile, hand-tuned, and will probably break. 
2. **Keep code clear.** Every non-obvious block should have a short comment explaining what it does and why.
3. **Be concise.** Helpful comments are great; a paragraph per line is not.

---

### Things To Mention

1. Keep inline comments short — one brief sentence is enough.
2. If you add a new shell command, add it to the command table in `shell/commands.c` and make sure it shows up in `help`.

---

## Outro

Thanks for supporting this small OS. Every contribution helps, no matter how small.
